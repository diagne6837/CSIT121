Lab05 Reflection

For this assignment, the focus was making my blog page responsive using CSS flexbox. I structured the layout so that on mobile devices, all blog posts appear in a single column format and on desk top screens, the page to a two column layout using media queries.it helps create a better user experience by adapting to different screen sizes. I was testing the page as I was going so I wouldn't accumulate too many syntax errors at the end. Overall, this assignment is very complicated and very technical but it gave me a better understanding of CSS grid and responsive design.
